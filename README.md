# 📈 基金定投计算器

一个简单实用的基金定投计划计算器，帮助你合理分配每月投资金额。

## 功能特点

- ✅ 支持自定义每月定投总金额
- ✅ 预设 5 只主流指数基金组合
- ✅ 自动计算每只基金的每月/每周定投金额
- ✅ 实时验证组合比例（必须等于 100%）
- ✅ 清晰的表格展示和投资建议

## 本地运行

```bash
# 安装依赖
pip install -r requirements.txt

# 运行应用
streamlit run app.py
```

## 部署到 Streamlit Cloud

1. 将此仓库推送到 GitHub
2. 访问 [share.streamlit.io](https://share.streamlit.io)
3. 点击 "New app"
4. 选择你的仓库和分支
5. Main file path 填写 `app.py`
6. 点击 "Deploy!"

## 使用示例

假设每月定投 5000 元，组合比例如下：
- 纳斯达克 100: 25%
- 标普 500: 25%
- 中证 A500: 20%
- 中证红利：15%
- 恒生科技：15%

计算结果：
- 每只基金每月定投金额自动计算
- 每周定投金额 = 每月金额 ÷ 4

## 定投策略建议

1. **定投时间**: 每月发工资后 1-3 天执行
2. **周投分配**: 可将月投金额分为 4 次，每周固定日期买入
3. **持有周期**: 建议 3-5 年以上长期持有
4. **再平衡**: 每半年检查一次比例，偏离超过 5% 时调整
